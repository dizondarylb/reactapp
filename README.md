# Node.js and React Take Home Test Instructions
Node.js and React

# Set-up
For Backend,
> cd api 

> npm install

> node server.js

- access it using http://localhost:5000 (change port if 5000 already used).

For Frontend
> cd client

> npm install

> npm start

- access it using http://localhost:3000 (change port if 3000 already used).

Enjoy!
